## BUG-01: Visual arrow icon misalignment inside generator cards

**Component:** UI / Cards Layout  
**Environment:** Chrome v126 (Desktop), Windows 11  
**Preconditions:** User is on the main page  
**Severity:** Low | **Priority:** Low  

### Description:
An arrow icon inside one of the generator cards is misaligned relative to adjacent cards, breaking the grid design consistency across the section.

### Steps to Reproduce:
1. Open the main page 
2. Scroll down to the "Functions / Generators" section.
3. Compare the visual alignment of the arrow icon in Card A with surrounding cards.

### Expected Result:
* All generator cards maintain uniform layout, spacing, and icon alignment.

### Actual Result:
* The arrow icon in the target card is visibly shifted vertically, breaking UI alignment.
