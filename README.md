## What this portfolio includes

This repository contains one main testing project:

### CSS Generators Testing Project

A web platform with multiple CSS and UI generators.

I tested:
- Navigation between tools
- UI/UX consistency
- Responsive behavior (mobile + desktop)
- Generator functionality
- Edge cases (large or unusual inputs)

---

## What I focused on during testing

While testing, I tried to simulate real user behavior:

- how a user moves between pages
- what happens when input values are changed
- how UI behaves on different screen sizes
- what breaks under unusual conditions

---

## Test artifacts

Each project contains structured QA artifacts:

- Bug Reports → documented issues found during testing
- Test Cases → structured test scenarios executed manually
- Checklists → functional and UI verification points
- Evidence → screenshots of bugs, workflow and testing process
